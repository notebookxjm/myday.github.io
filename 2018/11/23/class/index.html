类和对象学习笔记（小甲鱼）

Posted on 2018-11-23

类用class来声明，变量，函数等都可以在里面
class classname:
num=9
def func(self):
print(num)

类中的函数都有一个self的参数，相当与this指针，指向当前调用的对象

对象就是类的实例
obj=classname()
obj.func() #调用类里面的函数

init(self) #在创建一个对象的时候，会自动调用这个方法

class ball:
def init(self,name):
self.name=name
print(name)
def kick(self):
print(‘my name is %s’ % self.name)

a=ball(‘apple’)#初始化对象
a.kick()

类里面的属性和方法分为私有和公有
定义私有变量需要在变量名或函数名前加’‘2个下划线
class person:
name=’xxx’ age=23
def getname(self):
print(self.name)
def getage(self):
print(self.__age)

变为私有变量之后，这个变量不能从外部访问，用’.’来访问，只能从内部访问。

但是，私有变量其实是将变量改为了_类名变量名
所以
p=person()
p._personage
这样就可以访问私有变量。

类的继承
class DerivedClassName(BaseClassName):

class parent:
def say(self):
print(‘我是父类’)

class child(parent):
def csay(self):
print(‘我是子类’)
parent是父类，child是子类，子类会继承父类的方法和变量，并且可以调用
c=child()
c.say()
结果为：我是父类

如果子类定义了与父类同名的方法或属性，则会自动覆盖父类的方法和属性，对
父类不受影响

class child(parent):
def say(self):
print(‘我是子类’)
c=child()
c.say()
结果是：我是子类

如果想要父类的方法
1.调用未绑定的父类方法
class child(parent):
def say(self):
parent.say(self) #相当于parent.say(c)
print(‘我是子类’)

c=child()
c.say()
结果是：我是父类
我是子类

2.使用super()函数
class child(parent):
def say(self):
super().say()
print(‘我是子类’)

c=child()
c.say()
结果是：我是父类
我是子类

多重继承,有多个父类
class DerivedClassName(BaseClassName1,BaseClassName2,BaseClassName3):

class base1:
def say(self):
print(‘我是base1’)

class base2:
def say1(self):
print(‘我是base2’)

class base3(base1,base2):
def say2(self):
print(‘我是base3’)

base3的对象可以调用base1和base2的方法

b=base3()
b.say()
我是base1
b.say1()
我是base2

如果两个基类的函数名相同
class base2:
def say(self):
print(‘我是base2’)

class base3(base1,base2):
def say2(self):
print(‘我是base3’)

b=base3()
b.say()
我是base1

class base3(base2,base1):
def say2(self):
print(‘我是base3’)

b=base3()
b.say()
我是base2

#和两个基类的顺序有关
